SHORTCUT KEYS

    F10                     Selects toolbar menu.

    Ctrl+N                  New file
    Ctrl+O                  Open file
    Ctrl+S                  Save file

    F5                      Refresh

    Ctrl+X                  Cut
    Ctrl+C                  Copy
    Ctrl+V                  Paste
    Ctrl+A                  Select All

    Ins                     Add destination
    Del                     Remove destination
    F4                      Change destination

    F6                      Wake selected
    Shift+F6                Wake all
    Ctrl+W                  Quick wake


VERSION HISTORY

    3.00 (2011-08-28)

        o No registry writes will be performed if application is not installed.
        o Double-click now wakes computer instead of editing entry.
        o Broadcast address now can be host name (useful for waking over
          Internet).
        o Columns can be shown and hidden.
        o Lot of bug fixing.

    2.21 (2010-10-30)

        o Fixed bug that caused sending of empty error reports.

    2.20 (2010-10-12)

        o Bug fixing.
        o Added option to wait between waking computers.
        o Renamed console application to wol.exe (easier auto-complete).

    2.10 (2009-08-02)

        o Enhanced compatibility with Mono.
        o Fixed menu underlines.
        o Extension .magiwol is now associated with program.
        o Invalid open and save operations now go without crashing.

    2.00 (2009-06-30)

        o Rebuilt with new interface.
        o Added command-line interface (MagiWolCL.exe).
        o Full support for Windows Vista.
        o Support for copy/paste.
        o Import from computer names.
        o Changed hotkey assignment (F5, F6).
        o Multiple instances are allowed.

    1.20

        o Built on .NET Framework 2.0.

    1.10

        o Fixed Wake Checked button.
        o Added support for SecureOn.

    1.00 (2005-06-11)

        o First public release.
