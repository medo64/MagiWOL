SHORTCUT KEYS

    F10                     Selects toolbar menu.

    Ctrl+N                  New file
    Ctrl+O                  Open file
    Ctrl+S                  Save file

    F5                      Refresh

    Ctrl+X                  Cut
    Ctrl+C                  Copy
    Ctrl+V                  Paste
    Ctrl+A                  Select All

    Ins                     Add destination
    Del                     Remove destination
    F4                      Change destination

    F6                      Wake selected
    Shift+F6                Wake all
    Ctrl+W                  Quick wake
